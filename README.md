# W3-Assignment-LLM
## Project Setup
**Clone Repo:**
```
git clone https://github.com/Chy-Zaber-Bin-Zahid/W3-Assignment-LLM.git
```
**Change Directory:**
```
cd W3-Assignment-LLM.git/Scarp\ Tip/
```
**Run Scrapy:**
```
docker-compose up -d
```
**Change Directory:**
```
cd ../LLM
```
**Create .env File:**
```
echo "API_KEY='your_gemini_key'" > .env
```
**Run Django:**
```
docker-compose up -d
```
